## Summary

- **Sequential:** Steps execute one after another (slow)
- **Parallel:** Independent steps execute simultaneously (fast)
- **Speedup:** Ratio of sequential time to parallel time
- **Key:** Identify which steps can run in parallel (dependencies)
- **Benefit:** 3-10x faster for typical research workflows
